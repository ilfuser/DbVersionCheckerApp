#Простое приложение для определения версии базы данных

В решении добавлены два проекта: веб-сервис и приложение с пользовательским интерфейсом (UI).

Каждый проект создает отдельное независимое приложение, выполняющее свою роль: веб-сервис предоставляет 3 функции, GUI позволяет подключиться к веб-сервису и активировать методы.

Приложение и веб-сервис позволяют:
- открыть соединение с базой данных MS SQL, 
- поддерживать его открытым,
- получить информацию о версии базы данных,
- закрыть подключение к базе данных.

Приложение веб-сервис и GUI-приложение клиента с элементами управления.

**Для начала работы**

1. Установить путь к базе данных в web.config.
2. Убедиться, что путь к веб-сервису корректен в данном окружении в файле App.config

Веб-сервис содержит 3 метода + бонусный:
1. Connect
2. Version
3. Disconnect

**Особенности функционала**

Подключение осуществляется изолированно для каждого запущенного окна GUI,
сессия подключения для каждого окна помечается уникальным GUID.

При подключении GUID сохраняется на сервере, при отключении стирается из списка подключений на сервере.

При закрытии окна происходит авто-отключение.


Интерфейс пользователя содержит кнопки для соответствующих методов, а также поле отображения сервисных сообщений.

Сервисные сообщения отображаются в виде сырого JSON-объекта с ответом от сервера.

Установлен таймаут подключения в 30 секунд.
